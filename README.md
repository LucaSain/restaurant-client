# Restaurant client

A restaurant's digital menu.

## Tech stack:
+ FE: Vue3 and Nuxt
+ UI: TailwindCSS and DaisyUI
+ Realtime: Ably
+ DB: Upstash Redis
+ Storage Bucket: CloudFlare R2.

Serverside Rendered

## Env variables:
    UPSTASH_REDIS_REST_URL -> STRING
    UPSTASH_REDIS_REST_TOKEN -> STRING
